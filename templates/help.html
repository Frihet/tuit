Welcome to Tuit

Tuit is The Ultimate ITIL Toolbox. It is a set of software components designed to support ITIL based workflow for service managment. Tuit currently consists of the following components:

  • An issue handling system, based on Roundup
  • A CMDB based on FreeCMDB
  • A knowledge base, based on Foswiki.

Tuit Issue Managment

The Tuit issue handling component is an issue tracker losely based on Roundup. It is intended to be used to track incidents, problems and requests for change as they make their way through your IT organization.
